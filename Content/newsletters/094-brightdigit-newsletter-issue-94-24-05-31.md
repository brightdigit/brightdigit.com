<p>I’m working hard on finalizing the release of <a
href="https://getbushel.app" target="_blank"><strong>Bushel
v1.2.0</strong></a>, which comes with a host of new features and
improvements:</p>
<ul>
<li><p><strong>Overall Performance Improvements:</strong> Enhanced
performance to prepare for Swift 6, ensuring a smoother and faster
experience.</p></li>
<li><p><strong>WishKit Integration:</strong> Now you can request and
vote on new features directly within the app.</p></li>
<li><p><strong>Improved Document Management:</strong> Better tracking of
moved and deleted files for Machines and Libraries.</p></li>
<li><p><strong>Enhanced Error Tracking and User Feedback:</strong>
Improved systems to catch errors and collect your valuable
feedback.</p></li>
<li><p><strong>Brand New Machine Creation Dialog:</strong> A redesigned
interface for creating new machines, making the process more
intuitive.</p></li>
<li><p><strong>Improved Installation Screen:</strong> More descriptive
progress indicators to keep you informed during installations.</p></li>
</ul>
<p>Additionally, <a href="https://getbushel.app/">make sure to join the
Bushel TestFlight list.</a> <strong>My focus is on getting Bushel ready
for WWDC and the upcoming macOS version.</strong> Join the TestFlight <a
href="https://getbushel.app/">here</a>.</p>
<p><strong>Stay tuned for the official release
announcement!</strong></p>
