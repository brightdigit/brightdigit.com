<p><a href="https://www.serversideswift.info" target="_blank">The
conference</a> was a fantastic opportunity to dive deep into the world
of Server Side Swift and connect with fellow developers. <a
href="https://www.serversideswift.info/videos" target="_blank">All the
videos from the conference are published</a> but here are some
highlights from my favorite talks:</p>
<h3 id="swift-and-java-interoperability"><a
href="https://www.youtube.com/watch?v=wn6C_XEv1Mo"
target="_blank"><strong>Swift and Java
Interoperability</strong></a></h3>
<p>Ben Cohen and Tony Parker from Apple announced upcoming Java
interoperability for Swift. This groundbreaking development will allow
millions of server applications written in Java to transition to Swift,
mirroring Apple's own transition from C++ and Objective-C. It reminded
me of <a href="https://www.youtube.com/watch?v=hPSjbD3y_Bs"
target="_blank">my video on Objective-C transitioning</a> - exciting
times ahead!</p>
<h3 id="structured-concurrency-patterns"><a
href="https://www.youtube.com/watch?v=JmrnE7HUaDE&amp;t=15s"
target="_blank"><strong>Structured Concurrency
Patterns</strong></a></h3>
<p>Franz Busch from Apple shared invaluable insights on adopting
structured concurrency, along with best practices for building
composable libraries and applications using this paradigm.</p>
<h3 id="moving-things-to-server-side-swift"><a
href="https://www.youtube.com/watch?v=oJArLZIQF8w"
target="_blank"><strong>Moving</strong> <em><strong>Things</strong></em>
<strong>to Server Side Swift</strong></a></h3>
<p>Vojtech Rylko from <a href="https://culturedcode.com/things/"
target="_blank">Things</a> gave a fascinating talk about their
transition to using Swift on the server with AWS, providing real-world
examples of Server Side Swift in action.</p>
<h3 id="hummingbird-2"><a
href="https://www.youtube.com/watch?v=FHO_BfidQlQ"
target="_blank"><strong>Hummingbird 2</strong></a></h3>
<p>Joannis Orlandos introduced the new release of <a
href="https://github.com/hummingbird-project/hummingbird"
target="_blank"><strong>Hummingbird 2</strong></a>, a powerful Server
Side Swift framework that's pushing the boundaries of what's possible
with Swift on the backend.</p>
<h3 id="so-you-think-you-know-swift"><a
href="https://www.youtube.com/watch?v=smkRzwANNQ8"
target="_blank"><strong>So You Think You Know Swift</strong></a></h3>
<p>Nick Lockwood, creator of <a
href="https://github.com/nicklockwood/SwiftFormat"
target="_blank">SwiftFormat</a>, delved into lesser-known aspects of
Swift, offering insights that can help developers write more efficient
and elegant code.</p>
