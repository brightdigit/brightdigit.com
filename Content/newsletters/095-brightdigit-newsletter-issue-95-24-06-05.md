<ul>
<li><p><strong>Enhanced Performance:</strong> Significant improvements
to ensure a smoother and faster experience, setting the stage for Swift
6.</p></li>
<li><p><strong>WishKit Integration:</strong> Request and vote on new
features directly within the app, driving user-driven
innovation.</p></li>
<li><p><strong>Improved Document Management:</strong> Streamlined
management of machines and libraries, with better tracking of moved and
deleted files.</p></li>
<li><p><strong>Advanced Error Tracking and User Feedback:</strong>
Enhanced error tracking and a revamped feedback system for a more robust
user experience.</p></li>
<li><p><strong>Brand New Machine Creation Dialog:</strong> Redesigned
for a more intuitive and streamlined machine creation process.</p></li>
<li><p><strong>Improved Installation Screen:</strong> More descriptive
progress indicators, providing clear status updates during
installation.</p></li>
</ul>
